Esto son algunos de los ultimpos proyectos realizados:

Aplicación para la gestión de inspecciones realidada para Senso, esta aplicación tiene como funcion poder gestionar las inspecciones a realizar en las instalaciones o equipos de las instalaciones, dichas inspecciones son configurables en tiempo y perioricidad, las funcionalidades a utilizar son gestionadas por los permisos del usuario, tambien se dota de herramientas para dar de alta servicios tecnicos, instalaciones y equipos, asi como poder saber quien y cuando modifico que

Aplicación para la gestión de productos, presupuestos, pedidos, almacenes y control de la calidad, esta aplicación proporciona las herramientas basicas para la creación de productos y configuracion de sus tareas maestras, crear presupuestos y luego pedidos de venta con sus tareas y documentacion necesaria para su fabricación, gestión de los productos del almacen y gestionar el control de calidad, mediate la creacion de no conformidades, la creacion de pedidos

Plugin que enlaza Cios Works con SolidWork, este plugin conecta Cios Works con la aplicación de diseño SolidWorks, extrae los datos de los productos diseñados en SolidWork de forma automatica y permite desde el SolidWork poder configurar los productos con sus tareas maestras de fabricacion

Addin que enlaza Cios Works con el PDMWorks, esta addin conecta Cios Works con el gestor documental PDMWorks, permite la codificación de los productos, el clonado y que desde el PDMWorks acceder al Cios Works para poder configurar los productos con sus tareas maestras de fabricación

Aplicación para la gestión de citas, esta aplicación proporciona las herramientas necesarias para la gestión de citas, pudiendo elegir la fecha y poniendo los datos necesarios de la misma

Aplicación para la gestión de ventas en un establecimiento, esta aplicación proporciona las herramientas necesarias para la gestion de las ventas de un establecimientos, permitiendo la creación de productos y la gestión del almacén

Aplicación para la gestion empresarial realizada para Neograf Alvedro SLU, esta aplicación proporciona la gestión empresarial de una pequeña y mediana empresa, dispone de los modulos necesarios para la gestión de la misma (Presupuestos, Ventas, Compras, Almaceé, Facturación, Produccion, Calidad, ...)

Aplicacion para la gestión de un almacén logístico realizada para el grupo Kimak, esta aplicación proporciona las herramientas necesarias para la gestión de un almacen logístico destinado al almacenaje de productos de decoración
